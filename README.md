# Build openssl for android

[![Build Status](https://travis-ci.org/alvisisme/android-openssl.svg?branch=master)](https://travis-ci.org/alvisisme/android-openssl)

编译 Android 平台 arm64-v8a 架构的 openssl 静态库.

## 测试环境

* ubuntu16.04
* android-ndk-r13b
* openssl-1.0.2g

## 编译步骤

```shell
make
```
